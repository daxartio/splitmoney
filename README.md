# SplitMoney Lite

Lightweight browser app for shared expense tracking. All data is stored locally in the browser.

## Features

- Participant management (add, rename, activate/deactivate)
- Expenses with icon, payer, split helpers, and manual share editing
- Full ISO currency list + custom currency code input
- Last selected currency is saved and reused automatically
- Balances and settlement suggestions are calculated per currency
- Settlement tracking (debt repayments)
- Unified history with search
- CSV import/export with validation
- Offline-ready PWA (manifest + service worker)
